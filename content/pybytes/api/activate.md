**Activate**
----
  Activate device

**Method**
----
**pybytes.activate(activation_string)**


**Parameters**
----
| name  | Description   | is Required    | Default value
| ------------- |:-------------:|:-------------:|:-------------:|
| activation string   | generated by Pybytes  | Yes   | - |


**Example**
----
pybytes.activate("eyJhIjoiNTYzM...")
