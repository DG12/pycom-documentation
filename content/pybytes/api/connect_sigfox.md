**Connect Sigfox**
----
  Initialize Sigfox connection.

**Method**
----
**pybytes.connect_sigfox()**

**Example**
----
`pybytes.connect_sigfox()`

**Success Response**
----
    >> Connected using Sigfox. Only upload stream is supported
